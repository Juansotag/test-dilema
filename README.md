# Dilema Electoral: Encuentra tu Candidato

Este proyecto es una herramienta interactiva diseñada para ayudar a los ciudadanos colombianos a identificar cuál de los precandidatos a las **Elecciones Presidenciales de 2026** se alinea mejor con sus valores y propuestas.

## Características

- **Test de Afinidad**: Un cuestionario de 30 preguntas que abarca temas clave como Economía, Trabajo, Salud, Pensiones, Seguridad y Medio Ambiente.
- **Ranking Personalizado**: Al finalizar el test, se presenta un ranking de afinidad con los candidatos basado en las respuestas del usuario.
- **Conoce a tus Candidatos**: Espacio dedicado para explorar biografías detalladas (perfiles de dos párrafos) de cada candidato y sus posiciones frente a los dilemas planteados (consideraciones internas del GovLab).
- **Diseño Premium**: Interfaz moderna con efectos de glassmorphism, modo oscuro integrado y optimizada para una experiencia de usuario fluida.

## Tecnologías Utilizadas

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Diseño**: Google Fonts (Inter), Iconos SVG personalizados.
- **Data**: Archivo JSON para gestionar las preguntas y perfiles de candidatos.

## Instalación y Uso Local

Para ejecutar este proyecto en tu máquina local:

1. Clona el repositorio o descarga los archivos.
2. Abre una terminal en la carpeta del proyecto.
3. Inicia un servidor local simple. Por ejemplo, si tienes Python instalado:
   ```bash
   python -m http.server 8000
   ```
